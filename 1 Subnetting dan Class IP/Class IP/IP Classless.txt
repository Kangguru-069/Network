IP Classless
Pengalamatan IP Classless dapat diartikan menjadi "pengalamatan IP tanpa mengenal kelas". yaitu dengan cara menggunakan classless-inter Domain Routing (CIDR).
contoh :
- 172.16.24.1/24
- 10.10.1.1/28



IP Private
Digunakan untuk jaringan lokal
                  ip address range                number of address
24-bit block      10.0.0.0 - 10.255.255.255           16,777,216
20-bit block      172.16.0.0 - 172.31.255.255          1,048,576
16-bit block      192.168.0.0 - 192.168.255.255           65,536


IP Public
Digunakan untuk koneksi jaringan global
