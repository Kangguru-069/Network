Membuat Jaringan VLAN Server Dan Client
Menggunakan Router Cisco 3640

A. Setting Router (Router 1)
   R1#config terminal 
   R1(config)#interface Ethernet0/0
   R1(config-if)#no shutdown
   R1(config-if)#exit
   R1(config)#exit
   R1#show ip interface brief
   
   R1#config terminal
   R1(config)#interface Ethernet0/0
   R1(config-if)#ip address 172.48.105.1 255.255.255.0
   R1(config-if)#exit
   R1(config)#exit
   R1#show ip interface brief
   
B. Setting Domain Server (Router 2)
   R2#config terminal 
   R2(config)#interface Ethernet0/0
   R2(config-if)#no shutdown
   R2(config-if)#exit
   R2(config)#exit
   R2#show ip interface brief
   
   R2#config terminal
   R2(config)#interface Ethernet0/0
   R2(config-if)#ip address 172.48.105.5 255.255.255.0
   R2(config-if)#exit
   R2(config)#exit
   R2#show ip interface brief
   
C. Setting VLAN (Router 2)
   R2#config terminal 
   R2(config)#interface Ethernet0/1
   R2(config-if)#ip address 10.111.42.1 255.255.255.0
   R2(config-if)#exit
   R2(config)#exit
   R2#show ip interface brief
   
   R2#config terminal 
   R2(config)#interface Ethernet0/2
   R2(config-if)#ip address 10.111.48.1 255.255.255.0
   R2(config-if)#exit
   R2(config)#exit
   R2#show ip interface brief
