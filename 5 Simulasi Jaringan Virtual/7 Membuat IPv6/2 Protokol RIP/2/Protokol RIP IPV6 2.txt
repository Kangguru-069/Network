Membuat Jaringan menggunakan Protokol RIP (Routing Information Protocol)

Routing Information Protocol (RIP) adalah protokol vektor jarak yang menggunakan jumlah hop sebagai metrik utamanya. RIP mendefinisikan bagaimana router harus berbagi informasi ketika memindahkan lalu lintas di antara kelompok jaringan area lokal yang saling berhubungan.

Router yang digunakan : Router Cisco 3640

A. Setting Router-1 
   - Interface 2/0
     R1#config terminal     
     R1(config)#ipv6 unicast-routing 
     R1(config)#interface ethernet 2/0
     R1(config-if)#ipv6 address 7a5f:e50a:ff64:3a61:4e8c:51fd:493b:00001/64 atau (7a5f::1/64)
     R1(config-if)#ipv6 enable
     R1(config-if)#no shutdown
     R1(config-if)#exit
     R1(config)#write
     R1(config)#exit
     R1#show ipv6 interface brief
     R1#show ipv6 route
   
   - Interface 0/0
     R1#config terminal     
     R1(config)#ipv6 unicast-routing 
     R1(config)#interface ethernet 0/0
     R1(config-if)#ipv6 address 7a6d:6307:f7ea:480d:14f0:6ffd:75d8:00001/64 atau (7a6d::1/64)
     R1(config-if)#ipv6 enable
     R1(config-if)#no shutdown
     R1(config-if)#exit
     R1(config)#write
     R1(config)#exit
     R1#show ipv6 interface brief
     R1#show ipv6 route
     
   - R1#config terminal
     R1(config)#ipv6 router rip protokolName
     R1(config)#interface ethernet 2/0
     R1(config-if)#ipv6 rip protokolName enable
     R1(config-if)#interface ethernet 0/0
     R1(config-f)#ipv6 rip protokolName enable
     R1(config-if)#exit
     R1(config)#write
     R1(config)#exit
     R1#show ipv6 interface brief
     R1#show ipv6 route
